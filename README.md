# chem-spectra-client

ChemSpectra client side server.

This server integrates `react-spectra-editor` to view and edit chemical spectra (NMR, IR, and MS). File management, peaks output, and notifications are realized in this server.

### Usage

```
$ yarn install

$ yarn start
```
